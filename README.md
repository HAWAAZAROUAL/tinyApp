## Dependencies for TinyApp
##### - Node.js
##### - Express
##### - EJS
##### - bcrypt
##### - body-parser
##### - cookie-session

## Using TinyApp
##### Install all dependencies using ```npm install``` which can be found in the package.json file.
##### Run the app in the terminak using ```node express_server.js```.
